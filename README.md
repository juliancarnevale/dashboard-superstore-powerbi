# 📊 Dashboard de Ventas Superstore — Power BI

## Descripción
Dashboard interactivo de análisis de ventas construido con Power BI Desktop
sobre el dataset Superstore (9.800 pedidos, 4 años de datos).

## Vista previa

## Análisis incluidos
- Ventas totales por categoría (Furniture, Technology, Office Supplies)
- Ventas por región (West, East, Central, South)
- Distribución de ventas por segmento de cliente
- Top 10 subcategorías por ventas
- Segmentación interactiva por región

## Principales hallazgos
- Technology es la categoría con más ventas (36% del total)
- West es la región más rentable
- Consumer representa el 53% de las ventas totales
- Noviembre es el mes con mayor volumen de ventas (Black Friday)

## Transformaciones aplicadas
- Eliminación de columnas innecesarias (Row ID)
- Renombrado de columnas al español
- Detección automática de tipos de datos (fechas, números, texto)

## Herramientas utilizadas
- Power BI Desktop
- Power Query Editor
- DAX (medidas básicas)
